# Live Bakery 🍰

A full-stack web application for an online bakery management and ordering system built with Python, HTML, JavaScript, and CSS.

## 📋 Overview

Live Bakery is a comprehensive platform designed to streamline bakery operations and provide customers with an intuitive interface for browsing, ordering, and managing bakery products online.

## 🛠️ Tech Stack

- **Backend:** Python
- **Frontend:** HTML, JavaScript, Tailwind CSS
- **Database:** PostgreSQL / SQLite

## ✨ Features

- Product catalog and browsing
- Online ordering system
- User account management
- Order tracking
- Responsive web design

## 🚀 Getting Started

### Prerequisites

- Python 3.x
- pip (Python package manager)
- Modern web browser

### Installation

1. Clone the repository:
```bash
git clone https://github.com/aarchi-palikhel/Live_Bakery.git
cd Live_Bakery
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```


## 📁 Project Structure

```
Live_Bakery/
```

## 🎯 Main Features

- **Product Management:** Browse and explore bakery products
- **Shopping Cart:** Add items to cart and proceed to checkout
- **Order Placement:** Simple and secure ordering process
- **User Profiles:** Create accounts and manage order history
